# TCP Server

TCP-сервер для сохранения и получения данных датчиков.

## Возможности

- Сохранение записей датчиков (POST)
- Получение записей по имени датчика (GET)
- TCP-протокол с стоп-символом `$`
- Конфигурация через `.env`
- Graceful shutdown (SIGINT/SIGTERM)
- Retry-логика для сетевых операций
